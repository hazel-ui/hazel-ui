# Color

`Color` is an enum that provides us with all available colors used throughout the project.

## Usage

```tsx
import { Color } from "hazel-ui";

<p style={{ color: Color.royalBlue }}>My text</p>;
```

## Available colors

Click on a color to copy its hex code.

<ColorBlockContainer>
  {enumKeys(Color).map((colorName) => (

    />
  ))}
</ColorBlockContainer>
