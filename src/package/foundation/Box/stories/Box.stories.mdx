# Box

This component can be used as a wrapper to define the layout for
children components. It includes the Flexbox, Layout, Space, and
Typography utilities from [styled-system][1].

## Usage

```tsx
import { Box } from "hazel-ui";

function Component() {
  return (
    <Box display="flex" justifyContent="space-between">
      <div>1</div>
      <div>2</div>
    </Box>
  );
}
```
